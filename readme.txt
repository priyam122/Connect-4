Instructions:

 Player VS Player 
	o Click the “Run” button on the right top of the screen
	o Next users will be on the main menu of the game where they can choose what game mode they would like to play
	o Player 1 will start and be asked to select a column (1 -7)
	o Player 2’s turn will happen after Player 1’s turn
	o The game will continue going back and forth until a win or a tie is determined
	o The game will stop after a win or tie happens 
	o If it’s a tie the game will ask if both players would like to play again if not, then game ends.
 Player VS Computer
	o In this game mode it is the same as vs a player but instead the user will be playing against the computer
